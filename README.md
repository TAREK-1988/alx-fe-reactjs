# ALX React Fundamentals

This repository contains a React application built with Vite as part of the ALX Front-End ReactJS curriculum.

## Project Structure
- alx-react-app: Main React application

## How to Run Locally
1. Install dependencies:
   - npm install
2. Start the development server:
   - npm run dev
